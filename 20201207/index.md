# 养鱼生活


自从同事给我捎来了鱼，开始了每日的养鱼生活，提前步入老年生活。

最开始送来的一条小花鱼，因为喂食没有经验，导致涨死了；一条奶牛鱼，因为路上颠簸，伤到了鱼漂，到家时已经有点病症，然后经过一周的抢救，还是没能挽救回来。现在，是后来的两条鱼，一条小黑，一条小花。
